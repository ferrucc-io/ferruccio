## Developer Shortcuts

| | Command |  Shortcut      |
| :-------------: | :------------- |  :-------------: | 
| 📚 | New Tab | ⌘ + T | 
| 🖼 | New Window | ⌘ + N | 
| 🙅‍♂ | Close Tab | ⌘ + W | 
| 🔎 | Open the dev console | ⌘ + ⎇ + I | 
| ➡️ | Move to tab on the right | ctrl + ⇥ |
| ♻️ | Reload a page | ⌘ + R |
|  | Hard Reload a page | ⌘ + ⇧ + R |
